Triangle Operations

I modified this app to perform different triangle operations using the values of the sides input by the sliders.

Four Different Operations

Legal Triangle?

The legal triangle button tests if the values input by the user add up to the sides of a real triangle or not.


Area?

The Area button runs the input values through an equation that outputs the area.


Triangle Type?

The Triangle Type button returns whether the triangle is equilateral, isoceles, or scalene.


3:4:5 Triangle?

This button tests to see if the sides of the triangle are a ratio of 3:4:5.
